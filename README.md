# SecureGov Framework

## Overview
SecureGov Framework is an integrated cybersecurity project that combines security governance, risk management, vulnerability assessment, and compliance monitoring using open-source security tools.

## Features
-> Network Scanning (Nmap)
-> Packet Analysis (Wireshark)
-> Vulnerability Assessment (OpenVAS)
-> Vulnerability Verification (Metasploit)
-> Report Management (DefectDojo)

## Tools Used
-> Kali Linux
-> Nmap
-> Wireshark
-> OpenVAS (GVM)
-> Metasploit
-> DefectDojo
-> Docker

## Project Workflow
1. Network Scanning
2. Traffic Analysis
3. Vulnerability Assessment
4. Vulnerability Verification
5. Report Management
6. Risk Analysis
